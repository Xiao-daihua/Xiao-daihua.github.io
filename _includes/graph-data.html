{%- comment -%} graph-data.html Emits two JS globals used by both the Library graph and topic pages: window.TITLE_MAP : { "lowercase title" : "/url/" } (for [[wikilink]] resolution) window.GRAPH : { nodes:[...], links:[...] } (force graph + backlinks) A "link" from topic A -> topic B exists when A's raw content contains either a [[B Title]] wikilink or a literal href to B's url. Backlinks are just the reverse direction, computed client-side. {%- endcomment -%}
