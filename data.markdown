Welcome to my database of research papers on various topics in theoretical physics that I find interesting.

### What is this database about?

This database provides a structured way to organize research papers and make them easily accessible for learning a topic. Papers are organized by Topics, and all the topics are organized by tags and listed below.

### How to contribute?

One can use [this tool](bibtex-converter.html) to help convert the bibtex file into a markdown table that can be easily added to this database.
