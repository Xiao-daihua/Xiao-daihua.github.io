## Original Model 

String Net Model aka Levin-Wen Model was introduced in  
- [String-Net Condensation: A Physical Mechanism for Topological Phases](/papers/2005-levin/) 
as a lattice model that realizes a large class of 2D topological orders. The model is defined on a trivalent graph, where each edge is labeled by a string type (or particle type) and the vertices satisfy certain fusion rules. The Hamiltonian consists of two types of terms: vertex terms that enforce the fusion rules and plaquette terms that create and annihilate string nets. The ground state of the model is a superposition of all allowed string net configurations, and the excitations correspond to anyons.
